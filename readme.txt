For Setup:
----------
1. Create host, example: 'http://infuse_media.loc' .
2. Copy file to dir, or use git clone https://github.com/salionovich/infuse_media.git .
3. Create database and table please, use sql from file './create_table.sql'.
4. Change database connection, please edit file './banner.php':6-9
5. Open url http://infuse_media.loc/index1.html or http://infuse_media.loc/index2.html for test.
